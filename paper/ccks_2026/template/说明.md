# 你最需要注意的（高优先级）

## 1. 必须使用官方模板（非常重要）

文档明确要求：

* LaTeX 用 Springer LNCS 模板
* 不要自定义复杂格式
* 提交：

  * `.tex`
  * `.bib/.bbl`
  * 图片
  * 最终 PDF



你实际要做的：

```latex
\documentclass[runningheads]{llncs}
```

然后用官方 samplepaper 改。

---

# 2. 标题层级规范（很重要）

LNCS 对标题格式要求严格：

* 一级标题编号：

  ```text
  1 Introduction
  ```

* 二级：

  ```text
  2.1 Dataset
  ```

* 不允许：

  ```text
  0 Introduction
  ```

* 三级标题：
  run-in heading（段内标题）



你在写作时：

* 不要乱改 section 样式
* 不要自己手动加粗字号
* 用：

  ```latex
  \section{}
  \subsection{}
  \subsubsection{}
  ```

即可。

---

# 3. 图表规范（你最容易踩坑）

这是 LNCS 投稿最常见雷区之一。

Springer 明确要求：

## 图 caption 在下面

```latex
\caption{xxx}
```

## 表 caption 在上面



---

## 不允许把公式/表格当图片

也就是说：

❌ 截图公式
❌ Excel截图表格
❌ png版公式

必须：

* 表格用 LaTeX table
* 公式用 equation
* Word 用可编辑公式



---

## 图片尽量矢量图

推荐：

* PDF
* EPS
* SVG 转 PDF

不要：

* 模糊 png
* 截图曲线图

---

## 黑白打印兼容（特别重要）

Springer 会打印黑白版。

所以：

❌ 红绿曲线区分
❌ 浅色热力图

应该：

* 不同线型
* marker
* hatch

否则打印后看不清。



---

# 4. 公式编号规则

要求：

```latex
\begin{equation}
...
\end{equation}
```

编号：

```text
(1)
(2)
```

不能：

```text
(3.1)
```



---

# 5. 引用格式（非常重要）

LNCS 是数字引用：

```text
[1]
[3-5]
```

不是 author-year。



所以：

```latex
\bibliographystyle{splncs04}
```

这是必须的。

---

## DOI 尽量补齐

Springer 很强调 DOI：

```bibtex
doi={...}
```



这会影响最终出版质量。

---

# 6. 页数限制

文档写的是：

* Full paper：
  12–15 页
* Short paper：
  6–11 页



但：

⚠️ 真正生效的是会议 CFP 页数要求。

例如：

* 有些 LNCS 会议 12 页
* 有些 15 页
* 有些 references 不计页数

所以：

> 以会议官网 CFP 为准。

---

# 7. 作者信息不能后改（非常重要）

Springer 明确：

一旦最终提交：

* 不能改作者顺序
* 不能增删作者
* 不能换通讯作者



因此：

投稿前必须确认：

* 作者顺序
* 单位
* 邮箱
* ORCID

---

# 8. Appendix 放 References 前面

很多人会写：

```text
References
Appendix
```

但 LNCS 要求：

```text
Appendix
References
```



---

# 9. Disclosure of Interests（现在很重要）

现在 Springer 要求：

必须写：

```text
The authors have no competing interests to declare.
```



很多会议模板里已经有了。

---

# 10. 不要乱改模板（核心原则）

Springer typesetter 最怕：

* 自定义 spacing
* 自定义字体
* 手动调页边距
* squeeze references
* 手动缩小公式间距

因为最终会被强制改回去。



所以：

> 用默认模板写最安全。

---

# 对你（金融事件抽取方向）最实际的建议

你做 IE/KG/LLM 类论文时：

## 最容易被卡的问题：

### 1. 表格太宽

LNCS 单栏很窄。

解决：

```latex
\resizebox{\textwidth}{!}{...}
```

或者：

```latex
\small
```

---

### 2. Ablation 表太长

建议：

* 主结果放正文
* 大表放 appendix

---

### 3. 图颜色不清晰

一定：

* marker
* dashed line
* hatch

---

### 4. 算法框容易超页

推荐：

```latex
\usepackage{algorithm2e}
```

或者简化成伪代码。

---
