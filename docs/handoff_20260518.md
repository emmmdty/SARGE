## 6. 关键脚本

| 脚本 | 用途 |
|---|---|
| `scripts/infer_checkpoint.py` | 推理入口。支持 `--no-adapter`（baseline）、`--sample`（采样）、`--k`、`--limit` |
| `scripts/eval_three_tracks.py` | 三 track 评测。自动转换 pred 格式 + filter gold + 跑 legacy_doc2edag / unified_strict / docfee_official |
| `scripts/train_sft.py` | SFT 训练入口（生产级） |
| `scripts/train_lrd.py` | LRD 训练入口（W4-W7） |

### 推理命令示例

```bash
# greedy baseline
python -u scripts/infer_checkpoint.py --no-adapter --dataset DuEE-Fin-dev500 --limit 500 --k 1

# greedy with adapter
python -u scripts/infer_checkpoint.py --ckpt <adapter_path> --dataset ChFinAnn-Doc2EDAG --limit 500 --k 1

# sampling ablation
python -u scripts/infer_checkpoint.py --ckpt <adapter_path> --dataset DuEE-Fin-dev500 --limit 500 --sample
```

### 评测命令示例

```bash
python -B scripts/eval_three_tracks.py --run-root runs/<run_name> --dataset <dataset>
```
